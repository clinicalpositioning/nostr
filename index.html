What is Nostr?

Nostr, which stands for "Notes and Other Stuff Transmitted by Relays", is a simple, open protocol that enables a decentralized, censorship-resistant social network.

Key features of Nostr:

  • Uses public-key cryptography to sign and verify messages, ensuring authenticity
  • Relies on a network of relays to transmit messages, rather than a centralized server
  • Gives users control over their data and identity
  • Highly scalable and resistant to censorship
  • Can be used to build various apps, from social media to Bitcoin exchanges

With Nostr, users run a client locally and publish signed messages to relays. Subscribers to those relays are then notified of the messages. Users can connect to multiple relays for greater reach.

The protocol aims to solve issues like censorship and lack of user control that are common on mainstream social media platforms. Nostr is a distinct protocol from cryptocurrencies and blockchains, focusing purely on data transmission and communication.

Why Nostr?

Nostr leverages a simple, decentralized architecture to enable an open, resilient social network protocol that anyone can build on and no single entity can control or censor. It's an exciting development in the push for a more open internet.
